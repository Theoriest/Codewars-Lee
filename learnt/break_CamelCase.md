# Break Camel Case
Camel case is a way naming varibles by capitilizing the first letter in each word.
In the challenge you are supposed to break the different words in camel case string

## What I learnt
I learnt that characters also have methods in python
I only new of string methods and these intially lead me down the wrong approach when trying to find a solution.
On discovery i arrived at a very simple solution using the character method *isupper()* which returns a **Bool** True if character is capitilized  and False if character is Lower case.
